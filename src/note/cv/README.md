## 计算机视觉面临的挑战
* 人眼接受的视觉数据极其多，且大部分是冗余的
* 视皮层会压缩信息，大脑高级中枢只处理一部分抽象的数据，但仍比其他信息多至少两个数量级
* 大脑有约100亿个细胞，每个细胞有大量突触，处理能力非常强
* 人类可以通过积极的使用和训练对眼脑系统进行动态"编程"
