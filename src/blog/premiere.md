## 基础操作
| 操作  | 快捷键  | 说明  |
| --------- |----------| -----|
| 新建文本图层 | ctrl+T |   |
| 不启用 | shitf+E |   |
| 打标志 | M | 选中素材就在素材上打，不选中则在时间轴上打 |
| 标记一段时间 | Alt+拖动标志 |   |
| 移动图层 | Alt+up/down |   |
| 播放速度 | j,k,l | J后退（连按速率加倍）、K暂停播放，L前进（连按速率加倍）， J+K 减速后退， K+L减速前进 |
| 音频波峰放大 | Alt+滚轮 | 鼠标悬停于音频轨道(左侧控制栏) 用Alt+滚轮可放大，缩小 |
| 逐帧微调 | left,right | 按住Shift则一次5帧  |
| 删除素材和间隙 | Shift+del |   |
| 相机视角 | 0 |   |
